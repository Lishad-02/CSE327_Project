\doxysubsection{Detailed Description}
This is a polyfill for the Php\+Token class introduced in PHP 8.\+0. We do not actually polyfill Php\+Token, because composer might end up picking a different polyfill implementation, which does not meet our requirements.

Tokenize the given source code and return an array of tokens.

This performs certain canonicalizations to match the PHP 8.\+0 token format\+:
\begin{DoxyItemize}
\item Bad characters are represented using T\+\_\+\+BAD\+\_\+\+CHARACTER rather than omitted.
\item T\+\_\+\+COMMENT does not include trailing newlines, instead the newline is part of a following T\+\_\+\+WHITESPACE token.
\item Namespaced names are represented using T\+\_\+\+NAME\+\_\+\texorpdfstring{$\ast$}{*} tokens.
\end{DoxyItemize}
