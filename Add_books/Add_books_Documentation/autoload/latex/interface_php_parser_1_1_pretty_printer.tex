print\+Format\+Preserving (\begin{DoxyParamCaption}\item[{array}]{\$stmts}{, }\item[{array}]{\$orig\+Stmts}{, }\item[{array}]{\$orig\+Tokens}{}\end{DoxyParamCaption})}

Perform a format-\/preserving pretty print of an AST.

The format preservation is best effort. For some changes to the AST the formatting will not be preserved (at least not locally).

In order to use this method a number of prerequisites must be satisfied\+:
\begin{DoxyItemize}
\item The start\+Token\+Pos and end\+Token\+Pos attributes in the lexer must be enabled.
\item The Cloning\+Visitor must be run on the AST prior to modification.
\item The original tokens must be provided, using the get\+Tokens() method on the lexer.
\end{DoxyItemize}
